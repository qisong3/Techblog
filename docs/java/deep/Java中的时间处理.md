## 概述
自JDK1.0起，时间处理相关API就已经发布。不过由于设计不合理，并且没有考虑到并发，时区等，一直为人所诟病。

JDK1.1，Java新增了时区处理，但并发的问题继续保留了下来。为了避坑，三方库如JODA Time给出了自己的解决方案。

这样显然让，JDK1.8推出了全新的日期/时间处理API，解决了顽疾.
