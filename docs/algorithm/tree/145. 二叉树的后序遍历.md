## 题设
给定一个二叉树，返回它的 后序 遍历。

示例:

输入: [1,null,2,3]  
   1<br>
&nbsp;&nbsp;\\<br>
&nbsp;&nbsp;&nbsp;2<br>
&nbsp;&nbsp;/<br>
&nbsp;3 <br>

输出: [3,2,1]

进阶: 递归算法很简单，你可以通过迭代算法完成吗？
